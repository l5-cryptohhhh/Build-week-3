# Istruzioni per Claude Code su questo repository

Prima di iniziare qualsiasi lavoro su questo progetto, leggi **[CHECKPOINT.md](CHECKPOINT.md)**:
contiene lo stato del progetto, le decisioni tecniche prese (e perche'),
i limiti noti e i prossimi passi suggeriti dai collaboratori precedenti.

Dopo aver completato modifiche significative (nuova funzionalita', refactor
rilevante, fix di un bug non banale, cambio di una decisione architetturale),
aggiungi una voce datata in fondo alla sezione **Changelog** di
`CHECKPOINT.md` con un riepilogo breve, e aggiorna le altre sezioni del
file (stack, limiti noti, prossimi passi) se sono cambiate. Non e'
necessario chiedere conferma per questo aggiornamento: fa parte del normale
flusso di lavoro su questo repo.

Per la documentazione rivolta agli utenti/sviluppatori (installazione,
comandi, endpoint, credenziali demo) fa riferimento a [README.md](README.md).
